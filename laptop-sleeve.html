Formulas:

Template Width (TW) = W + 2(H+P) + S

Template Length (TL) = (L+P)(2) + (H+P)(4) + (L+P)(0.5)

Taper Angle Offset (TAO) = 0.10(TW)

Where P = padding (0.25" / 0.635cm)

Where S = seam allowance (0.5" / 1.27cm)
